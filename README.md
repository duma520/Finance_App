## 系统简介
人民币收支管理系统是一款功能全面的个人/家庭财务管理软件，帮助用户记录收入、支出、借款和还款等财务活动。系统采用直观的界面设计和强大的数据分析功能，适合各种层次的用户使用。

**核心特点**：
- 多用户支持，保护个人隐私
- 借款还款管理，清晰记录借贷关系
- 智能分类统计，直观了解资金流向
- 数据备份恢复，保障财务安全
- 拼音首字母搜索，中文输入更便捷

## 基础功能
### 1. 记录收支
- **收入**：工资、奖金、投资收益等
- **支出**：餐饮、购物、交通等日常消费
- **借款**：借出款项给他人
- **还款**：收回借款或偿还债务

**示例**：
- 记录工资收入：类型选"收入"，类别选"工资"，输入金额
- 记录午餐支出：类型选"支出"，类别选"餐饮"，输入金额

### 2. 查看记录
- 表格形式展示所有交易
- 支持按类型、分类、时间筛选
- 颜色区分不同类型交易（绿色收入，红色支出）

### 3. 简单统计
- 收支总额对比
- 净收入计算
- 分类支出占比

---

## 专业功能
### 1. 借款管理
- 记录借款金额、日期和描述
- 跟踪还款进度
- 标记借款状态（待还款/已结清）

**专业提示**：
- 关联借款与还款，系统自动计算剩余待还金额
- 可部分还款，系统会累计还款金额

### 2. 高级筛选
- 组合筛选（类型+分类+时间范围）
- 拼音首字母搜索（输入"cy"可找到"餐饮"）
- 自定义日期范围

### 3. 数据管理
- 自动备份（每小时/每天/每周）
- 手动备份
- 数据恢复
- 操作撤销（最多10步）

### 4. 分类管理
- 自定义收入/支出分类
- 批量管理分类
- 分类使用统计

## 使用场景举例
### 场景1：家庭月度预算
1. 记录所有家庭成员收入
2. 设置固定支出分类（房贷、水电费等）
3. 月末查看"分类统计"，分析超支项目

### 场景2：朋友间借款
1. 记录借款：类型"借款"，输入金额和描述
2. 朋友还款时：类型"还款"，选择关联借款
3. 系统自动更新借款状态

### 场景3：自由职业者税务准备
1. 全年记录业务收入和支出
2. 导出特定时间段数据
3. 使用分类统计计算可抵扣支出

### 场景4：学生理财
1. 记录每月生活费收入
2. 分类记录各项支出（学习、娱乐等）
3. 设置预算提醒避免超支

## 安装与启动
### 系统要求
- Windows/MacOS/Linux
- Python 3.6+
- 推荐配置：4GB内存，100MB硬盘空间

### 安装步骤
1. 下载程序包
2. 解压到指定目录
3. 运行`Finance_App.py`

**注意**：首次运行会自动创建数据库和备份目录

## 详细使用指南
### 1. 用户管理
- 首次使用创建用户
- 支持多用户切换
- 用户数据独立存储

### 2. 记录交易
1. 选择交易类型（收入/支出/借款/还款）
2. 输入金额（支持小数点）
3. 选择分类（可自定义）
4. 填写描述（可选）
5. 设置日期（默认当天）
6. 点击"添加记录"

### 3. 数据筛选
- **快速搜索**：输入关键字搜索描述、金额或分类
- **拼音搜索**：输入分类拼音首字母（如"gw"找"购物"）
- **高级筛选**：组合类型、分类、状态和时间范围

### 4. 统计分析
- 选择统计时间段
- 查看收支趋势图
- 分析分类占比
- 导出统计结果（截图或复制文本）

### 5. 数据安全
- **自动备份**：设置每小时/每天/每周自动备份
- **手动备份**：重要操作前建议手动备份
- **数据恢复**：从备份文件恢复数据
- **撤销操作**：最多可撤销最近10次操作

## 常见问题解答
### Q1: 如何迁移数据到新电脑？
A: 复制整个程序目录和数据库文件（`finance_用户名.db`）

### Q2: 借款和支出有什么区别？
A: 借款需要后续还款，系统会跟踪状态；支出是一次性消费记录

### Q3: 分类可以修改吗？
A: 可以，通过"编辑"→"管理分类"添加/删除分类

### Q4: 数据会上传到网络吗？
A: 不会，所有数据本地存储，确保隐私安全

### Q5: 如何设置预算提醒？
A: 当前版本支持统计功能，可通过定期查看分类支出来监控预算
