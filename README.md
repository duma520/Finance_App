## 系统概述
**人民币收支管理系统**是一款功能完善的个人/家庭财务记账工具，由杜玛开发，采用MIT开源许可证。系统支持多用户、数据隔离存储，提供智能分类管理、统计分析、数据备份与恢复等功能。

### 核心功能
- 记录收入、支出、借款、还款等各类财务交易
- 多用户支持，数据隔离存储
- 智能分类管理，支持自定义分类
- 强大的统计分析和报表功能
- 数据备份与恢复机制
- 借款还款跟踪管理
- 拼音首字母快速搜索
- 自动定期备份数据
- 操作历史撤销功能

## 适用人群

### 个人用户
- **学生**：记录零花钱、奖学金、生活开支
- **上班族**：管理工资、日常消费、储蓄计划
- **家庭主妇/主夫**：统筹家庭收支，规划家庭预算
- **自由职业者**：追踪项目收入，管理业务支出

### 专业人士
- **财务人员**：辅助个人财务管理，分析收支结构
- **小微企业主**：简易账务管理，现金流监控
- **投资者**：记录投资收益，分析投资回报率

### 特殊需求群体
- **债务管理者**：跟踪借款还款状态，计算待还金额
- **预算制定者**：基于历史数据分析制定未来预算
- **税务申报者**：分类记录可作为税务申报的辅助依据

## 基础使用指南

### 1. 首次使用
1. **启动程序**：双击运行程序
2. **选择用户**：从已有用户中选择或创建新用户
3. **主界面介绍**：
   - 顶部：搜索和筛选区域
   - 中部：交易记录表格
   - 底部：新增记录表单

### 2. 添加记录
1. 选择交易类型（收入/支出/借款/还款）
2. 输入金额（正数）
3. 选择分类（可自定义）
4. 填写描述（可选）
5. 设置日期（默认为当天）
6. 点击"添加记录"按钮

**示例**：记录一顿午餐花费
- 类型：支出
- 金额：35
- 分类：餐饮
- 描述：公司附近牛肉面
- 日期：今天

### 3. 查看数据
- **表格视图**：所有记录按时间排序显示
- **统计分析**：点击"统计分析"标签页查看图表
- **快速搜索**：在搜索框输入关键字或拼音首字母

### 4. 备份数据
- 自动备份：系统每小时自动备份一次
- 手动备份：通过"文件"→"备份"→"手动备份"

## 专业功能详解

### 1. 借款还款管理
**专业应用场景**：适用于有借贷关系的个人或小微企业

**操作流程**：
1. 添加借款记录：
   - 类型：借款
   - 金额：借款总额
   - 分类：其他（可自定义）
   - 状态：pending（待还款）

2. 添加还款记录：
   - 类型：还款
   - 选择关联的借款记录
   - 系统自动计算剩余待还金额

**专业提示**：系统会自动跟踪每笔借款的还款进度，当还款总额≥借款金额时，状态自动变为"已结清"。

### 2. 高级统计分析
**功能亮点**：
- 多维度分析：按时间、分类、类型等多角度统计
- 自定义时间范围：精确到日的统计区间
- 可视化呈现：表格+文字结合的报表形式

**专业指标**：
- 净收入 = 总收入 + 总还款 - 总支出 - 总借款
- 收支比 = 总收入 / 总支出
- 债务比率 = 待还款 / 月收入

### 3. 数据安全机制
1. **自动备份**：每小时备份一次，保留最近备份
2. **手动备份**：用户可随时创建备份点
3. **恢复功能**：可从任意备份点恢复数据
4. **操作撤销**：支持最多10步操作撤销

### 4. 分类管理系统
**专业管理建议**：
1. 建立符合个人需求的分类体系
   - 收入分类示例：工资/奖金/投资回报/其他
   - 支出分类示例：固定支出/可变支出/投资支出

2. 分类层级设计（通过命名实现）：
   - 一级分类：餐饮
   - 二级分类：餐饮-早餐、餐饮-午餐、餐饮-晚餐

## 行业应用案例

### 1. 自由职业者应用
**场景**：独立设计师管理项目收支

**使用方法**：
1. 为每个客户创建专属分类
2. 记录项目收入时，在描述中注明项目名称
3. 记录项目相关支出（如软件订阅、设备采购）
4. 月末使用统计功能计算各项目净利润

### 2. 小微企业应用
**场景**：小型零售店简易账务管理

**使用方法**：
1. 收入分类按产品线设置
2. 支出分类包括：进货/租金/人工/营销等
3. 使用借款功能记录短期资金周转
4. 利用统计功能分析各产品线利润率

### 3. 家庭财务管理
**场景**：三口之家年度财务规划

**使用方法**：
1. 设置家庭共同账户
2. 分类包括：日常开支/教育/医疗/旅游等
3. 每月底使用"余额"功能记录账户结余
4. 年终统计各类支出占比，制定新年预算
